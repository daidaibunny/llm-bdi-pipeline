\begin{abstract}
We study the automatic synthesis of domain-level lifted AgentSpeak(L) plan
libraries from Planning Domain Definition Language domains and small sets of
achievement-goal training problems. The generated library is not a trace for a
single problem and is not tied to a query. Instead, its plan heads are lifted
predicate achievement goals and its bodies call declared primitive actions or
other predicate goals. The method combines goal-conditioned policy sketches,
modular predicate-goal rules, an answer-set selection problem, and a conservative
AgentSpeak(L) compiler. The current scope is positive conjunctive achievement
goals; temporal goals are handled by a separate controller and are outside this
paper core.
\end{abstract}

\section{Introduction}

Classical planners can generate a plan for a concrete problem, but a
Belief-Desire-Intention agent requires a reusable plan library. AgentSpeak(L)
provides a logical language for such agents \cite{Rao1996AgentSpeak}. This
paper targets the gap between generalized planning and executable agent
programs: given a \pddl{} domain and a small set of training problems,
synthesize a lifted \asl{} library that can solve new achievement-goal problems
from the same domain without asking a planner for a full trace at run time.

The method follows the view that many useful planning domains share a compact
subgoal structure expressible by sketches and generalized policies
\cite{Drexler2022LearningSketches,Drexler2024SubgoalSketches,
Bonet2024GeneralPoliciesWidth}. We adapt this view to \asl{} by making the
output modular. Atomic modules implement predicate goals such as
\texttt{clear(X)} or \texttt{on(X,Y)}. A separate top-level composer orders
conjunctive goals using read-only goal descriptors such as
\texttt{goal\_on(X,Y)}. The descriptors specify the current problem goal, while
the achievement-goal heads remain ordinary \pddl{} predicates.

The contribution claimed here is deliberately bounded. We do not claim universal
generalized-planning completeness for arbitrary \pddl{} domains. We claim a
bounded-class synthesis procedure for libraries expressible as
goal-conditioned modular policy sketches with safe lifted feature bindings.
Unsupported cases are rejected with diagnostics instead of being compiled into
unsafe plans.

\section{Current Scope and Exclusions}
\label{sec:scope}

The current method supports positive conjunctive achievement goals. Negative
goals, disjunctive goals, derived predicates, conditional effects, and numeric
fluents require additional semantics before they can be part of the paper
claim. Temporal extended goals are handled by a future deterministic finite
automaton controller that calls the achievement-goal library; they are not part
of the current algorithm.
